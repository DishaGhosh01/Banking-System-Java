# Banking System (Java)
  ## Description : 
    It is a simple console-based Banking system built using Java (Basics).It allows user to create  accounts , deposit money, withdraw money and check their current balance.
  ## Features:
  - **Create Account**.
  - **Deposit**
  - **Withdraw**
  - **Check balance**
  ## Technologies Used:
      -Java(Basics).
      -OOP Concepts.
      -ArrayList.
  ## How To Run:
    1.javac BankAccount.java file and  BankApp.java file.
    2.java BankApp.java file.
  ## Learning:
    - Object Oriented programming.
    - How to use ArrayList.
    - Building a real-world console application;
